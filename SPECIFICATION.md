# Specification for CSUAI Demo

## Overview
This is a simple Vite+React demo application. The main page displays a heading and a short informational paragraph.

## Current UI
- **Heading**: "CSUAI Demo"
- **Paragraph**: "You can make any changes you want to this website directly through the chat."

## Future Changes
- Users can request UI text modifications, component additions, or style updates via chat.
- All changes will be committed directly to the repository.
